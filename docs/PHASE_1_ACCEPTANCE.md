# Phase 1 Acceptance Criteria

Phase 1 qabul qilingan hisoblanadi, agar:

- [x] ikki biznes yo‘nalishi va ularning ustuvorligi hujjatlashtirilgan;
- [x] checkout emas, lead/appointment modeli tasdiqlangan;
- [x] payment-at-branch qarori qayd qilingan;
- [x] price visibility variantlari belgilangan;
- [x] lead → DB → outbox → Telegram → admin oqimi belgilangan;
- [x] appointment asosiy status va jarayoni belgilangan;
- [x] public va admin sitemap mavjud;
- [x] modular monolith arxitekturasi tanlangan;
- [x] migratsiya review-first va idempotent ekani belgilangan;
- [x] joriy sayt bo‘yicha tashqi audit cheklovlari qayd qilingan;
- [x] keyingi implementation backlog dependency tartibida yozilgan;
- [x] source discovery va extraction uchun ishga tushiriladigan skriptlar yaratilgan.
